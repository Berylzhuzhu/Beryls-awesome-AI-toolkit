## 🚀 快速开始

### 1️⃣ 选择你需要的 Skill
浏览 `skills/` 文件夹，找到符合你需求的工具。

### 2️⃣ 复制 Prompt
打开对应的 `.md` 文件，复制完整的 prompt 内容。

### 3️⃣ 使用
粘贴到你的 AI 助手（Claude、ChatGPT 等）中开始使用。

### 4️⃣ 自定义
根据文档说明，调整参数以适应你的具体需求。
